# 📊 MySQL Practice – SQL Fundamentals

## 📌 Overview

This repository contains a collection of MySQL practice queries designed to build a strong foundation in database management. It covers essential SQL concepts such as database creation, table operations, data insertion, constraints, and data analysis.

The project is ideal for beginners who want to understand how relational databases work and how to interact with them using SQL.

---

## 🚀 Features

* Create and manage databases
* Create tables with structured schema
* Insert and retrieve data
* Apply constraints (PRIMARY KEY, UNIQUE, NOT NULL, CHECK)
* Use foreign keys to build relationships
* Modify tables using ALTER commands
* Filter data using WHERE clause
* Sort and limit results
* Perform data analysis using aggregate functions
* Group and filter data using GROUP BY and HAVING

## 🛠️ Technologies Used
